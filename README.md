# My Portfolio

A modern, animated one-page portfolio website showcasing my frontend development skills and projects.

## Features
- Animated gradient hero with floating background blobs
- Typing effect for role titles
- Smooth scroll navigation with active link highlighting
- Scroll-triggered reveal animations
- Animated skill progress bars
- Responsive project cards with tech tags
- Contact form
- Fully responsive design

## Tech Stack
- HTML5
- CSS3 (Animations, Grid, Flexbox)
- Vanilla JavaScript (DOM Manipulation, IntersectionObserver)

## How to Run
Open `index.html` in any browser.
